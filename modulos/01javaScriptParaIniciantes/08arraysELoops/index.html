Arrays e Loops

Array

É um grupo de valores geralmente relacionados. Servem para guardar diferentes valores em uma única variável.

                
                    var videoGames = ['Switch', 'PS4', 'XBox'];
                    videoGames[0] // Switch
                    videoGames[2] // Xbox

Acesse o elemento "n" da array utilizando array[n-1].
