JavaScript para Iniciantes

Objetos

Objetos

Conjunto de variáveis e funções que são chamadas de propriedades e métodos. São instância de criadas a partir de uma classe. 


            var pessoa = {
                nome: 'André',
                idade: 28,
                profissao: 'Designer',
                possuiFaculdade: true,
              }
              
              pessoa.nome; // 'André'
              pessoa.possuiFaculdade; // true

Propriedades e métodos consistem em nome (chave) e valor.

Métodos

É uma propriedade que possui uma função no local de seu valor.

          
            var quadrado = {
              lados: 4,
              area: function(lado) {
                return lado * lado;
              },
              perimetro: function(lado) {
                return this.lados * lado;
              },
            }
            
            quadrado.lados; // 4
            quadrado.area(5); // 25
            quadrado.perimetro(5); // 20

Organizar o código

Objetos servem para organizar o código em pequenas partes reutilizáveis, bem como implementar o paradigma da Programação Orientada a Objetos(POO).

          
            Math.PI; // 3.14
            Math.random(); // número aleatório

            var pi = Math.PI;
            console.log(pi); // 3.14

Math é um objeto nativo de JavaScript, como é o console que utilizamos para exibir nossas mensagens em conjunto com o método log().

Um objeto é criado utilizando as chaves {}.

          
            var carro = {};
            var pessoa = {};

            console.log(typeof carro); // 'object'

Palavra-chave This

this irá fazer uma referência ao próprio objeto.

          
            var height = 120;
            var menu = {
              width: 800,
              height: 50,
              metadeHeight() {
                return this.height / 2;
              }
            }

            menu.metadeHeight(); // 25
            // sem o this, seria 60

this irá retorar o próprio objeto.

Protótipo e herança

O objeto herda propriedades e métodos do objeto que foi utilizado para criar o mesmo. Herda esses métodos da classe pai.

          
            var menu = {
              width: 800,
            }
            
            menu.hasOwnProperty('width') // true
            menu.hasOwnProperty('height') // false

hasOwnProperty é um método de Object.
