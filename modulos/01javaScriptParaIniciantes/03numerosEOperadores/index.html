Números e Operadores

Números

var idade = 28;

var gols = 1000;

var pi = 3.14; // ponto para decimal

var exp = 2e10; // 20000000000

Precisão para até 15 digitos.

Operadores em Strings

var soma = '100' + 50; // 10050, + força o número 50 a se tornar uma string.

var subtracao = '100' - 50; // 50, - força a string a se tornar um número.

var multiplicacao = '100' * '2'; // 200, * força as strings a se tornarem números

var divisao = 'Comprei 10' / 2; // NaN (Not a Number)

É possível verificar se uma variável é NaN ou não com a função isNaN();

NaN = Not a Number

var numero = 80;

var unidade = 'kg';

var peso = numero + unidade; // '80kg'

var pesoPorDois = peso / 2 // NaN (Not a Number)
