Tipos de dados

Sete tipos de dados

Todos são primitivos, exceto o objeto.

String, Number, Boolean, Undefined, Null, Symbol, Object

var nome = 'André'; // String

var idade = 28; // Number

var possuiFaculdade = true; // Boolean

var time; // Undefined

var comida = null; // Null

var simbolo = Symbol() // Symbol

var novoObjeto = {} // Object

Primitivos são dados imutáveis.

Verificar o tipo de dado:

'typeof nomeDaVariavel;'

Template String

Usado para acessar valores de uma variável dentro de uma String.

Exemplo: 'var gols = 10; var frase = `Romário fez ${gols} gols.`;'
